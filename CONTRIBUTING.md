# Contributing to content-audit-skill

PRs welcome. This skill covers post-publication content maintenance.

## What belongs here

- Compliance checks for published content
- Freshness and decay detection logic
- LLM readability testing patterns
- Re-test scheduling and tracking
- Citation rate monitoring approaches

## What belongs elsewhere

- Pre-publication optimisation: [seo-geo-skill](https://github.com/arturseo-geo/seo-geo-skill)
- Content creation pipeline: [content-pipeline-skill](https://github.com/arturseo-geo/content-pipeline-skill)
- Fact-checking methodology: [grounded-research-skill](https://github.com/arturseo-geo/grounded-research-skill)

## Guidelines

- Keep reference files practical and actionable
- Include verification commands (grep/curl patterns) where possible
- Test against real published pages before submitting
- Follow existing markdown structure
